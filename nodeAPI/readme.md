# ⚙️ API de Usuários

API backend desenvolvida para gerenciamento de usuários, com operações de cadastro, listagem e edição de dados.

</div>

---

<strong>Visão Geral do Projeto</strong>

<table>
  <tr>
    <td width="60%">

Este projeto consiste em uma API REST desenvolvida com o objetivo de praticar conceitos fundamentais de backend, incluindo manipulação de dados e comunicação via requisições HTTP.

A aplicação permite cadastrar, listar e editar usuários, utilizando um arquivo JSON como base de dados para simular persistência.

O projeto também foi pensado para integração com um frontend, onde a API é consumida para exibição e manipulação das informações.
